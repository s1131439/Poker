# Poker
### 遊戲目標
1. 在五張卡牌內，盡可能組合出更具勝算的五張牌型
2. 總資金規零時，遊戲結束

   ---
   
### 操作步驟
1. 輸入押注金額（原始金額定為1000000）
2. 按下「押注」按鈕，確認押注金額
3. 按下「發牌」按鈕，進行發牌
4. 若需要換牌，則點擊該卡牌，將其翻至背面，若要取消選擇該卡牌，再次點擊即可
5. 確認要更換的卡牌後，按下「換牌」按鈕
6. 按下「判斷牌型」按鈕，確認是否獲勝

   ---
   
### 操作體驗
1. 若輸入錯誤押注內容會彈出提示視窗
2. 總金額在金額變動時，有動畫設計
3. 設有「連勝、連敗」紀錄：
   - 連勝 3 場：顯示金色  
   - 連勝 5 場：顯示紅色  
   - 連勝 7 場：顯示深紅色  
   - 連敗 3 場：顯示藍色  
5. 若總金額為0，則判斷「破產」
